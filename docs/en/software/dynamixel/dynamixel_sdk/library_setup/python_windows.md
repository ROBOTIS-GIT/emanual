# [Build and Install the SDK](#build-and-install-the-SDK)

- Open the `{$Dynamixel_SDK}/python` folder.
- Right click on the folder and select `Open in terminal`.
- Build and install Dynamixel_SDK through below command.
```bash
$ pip install .
```
