| Output                 | 20W                                                                                                                 |
| Operating Temperature  | 5&deg;C ~ 55&deg;C                                                                                                  |
| Nominal Voltage        | 24V                                                                                                                 |
| Command Signal         | Digital Packet                                                                                                      |
