# [개요](#개요)

![img](/assets/images/parts/motor/l_speed_geared_motor_product.jpg)     

> 저속 감속 모터

로보티즈 드림(ROBOTIS DREAM) 1단계에서 사용되는 모터 입니다.

- `제품 호환 가이드` [제어기 호환표]

# [제품 사양](#제품-사양)

- 무게 : 15g
- 크기 : 18mm x 36mm x 24mm
- 기어비 :  70 : 1
- 속력 : 150 RPM(at 3.7V)
- 특징
  - 양쪽 축 출력
  - 안전장치(클러치) 장착
