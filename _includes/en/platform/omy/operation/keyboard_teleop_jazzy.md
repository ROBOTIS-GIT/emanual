Control the manipulator (simulation or hardware) using your keyboard
```bash
ros2 run open_manipulator_teleop omy_f3m_teleop
```
**Joint Control**
- `1` / `q` - Joint 1
- `2` / `w` - Joint 2
- `3` / `e` - Joint 3
- `4` / `r` - Joint 4
- `5` / `t` - Joint 5
- `6` / `y` - Joint 6

**Gripper Control**
- `o` - Open gripper
- `p` - Close gripper
