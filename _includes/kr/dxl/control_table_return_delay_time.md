
제어기로부터 Instruction Packet을 받은 후, Status Packet을 반환하기까지 걸리는 시간입니다.  
0~254 (0xFE) 까지 사용 가능하며 단위는 2μsec 입니다.  
예를 들어, 값이 10일 경우 20 μsec 만큼 시간이 지난 후에 Status Packet을 응답합니다.

|  단위   | 값의 범위 |                  설명                   |
|:-------:|:---------:|:---------------------------------------:|
| 2[μsec] |  0 ~ 254  | 기본값 ‘250’(500[μsec]), 최대 508[μsec] |
